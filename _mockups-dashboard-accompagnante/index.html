Dashboard accompagnante — 3 directions

Trois manières de repenser le dashboard, toutes orientées chaleureux + minimaliste.

1Le journal de bord

Une page qui se lit comme une lettre. Hiérarchie temporelle (aujourd'hui / cette semaine / votre présence). Pas de cartes, du texte qui respire.

2Le foyer

Portrait au centre, 3 chiffres sobres, beaucoup d'espace. L'utilisatrice se reconnaît, le reste s'efface.

3La conversation

Le dashboard te parle. Une seule action en avant, ton humain. Le reste est résumé en un paragraphe rassurant.
